# 나는야 포켓몬 마스터 이다솜

### [ 문제 정보 ]
> **백준**: https://www.acmicpc.net/problem/1620
> 
> **난이도**: 1
>
> **유형**: #자료구조


### [ 문제 풀이 ]
```C++
#include <bits/stdc++.h>
using namespace std;


int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);
    int a,b;
    cin >> a>> b;

    map<int, string> indexData;
    map<string, int> stringData;
    for (int i=1 ; i<=a ; i++) {
        string chrt;
        cin >> chrt;
        indexData.insert(make_pair(i, chrt));
        stringData.insert(make_pair(chrt, i));
    }
    for ( int i=0 ; i<b ; i++) {
        string nameOrNumber;
        cin >> nameOrNumber;
        if (isdigit(nameOrNumber[0])) { // 인덱스인 경우
            auto it = indexData.find(stoi(nameOrNumber));
            cout << it->second << "\n";
        } else { // 이름인 경우
            auto it = stringData.find(nameOrNumber);
            cout << it->second << "\n";
        }
    }
}
```
ios_base::sync_with_stdio(false);<br>cin.tie(NULL);<br>이걸 안해줘서 계속 시간초과가 남
