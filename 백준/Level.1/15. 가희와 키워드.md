# 가희와 키워드

### [ 문제 정보 ]
> **백준**: https://www.acmicpc.net/problem/22233
> 
> **난이도**: 1
>
> **유형**: #자료구조


### [ 문제 풀이 ]
```C++
#include <bits/stdc++.h>
#include <sstream>
using namespace std;


int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);
    int a,b;
    cin >> a>> b;
    set<string> wordList;
    for (int i=0 ; i<a; i++) {
        string word;
        cin>> word;
        wordList.insert(word);
    }
    for (int i =0 ; i<b; i++) {
        string words;
        cin>> words;
        istringstream ss2(words);
        string token;
        while (getline(ss2, token,',' )) {
            auto it = wordList.find(token);
            if (it != wordList.end()) {
                wordList.erase(it);
            }
        }
        cout << wordList.size() << "\n";

    }
    
}
```
