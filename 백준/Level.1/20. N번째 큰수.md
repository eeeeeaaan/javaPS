# N번째 큰수

### [ 문제 정보 ]
> **백준**: https://www.acmicpc.net/problem/2075
> 
> **난이도**: 1
>
> **유형**: #자료구조


### [ 문제 풀이 ]
```C++
#include <iostream>
#include <queue>
#include <vector>

int main() {
    std::ios_base::sync_with_stdio(false);
    std::cin.tie(NULL);

    int n;
    std::cin >> n;

    // 최소 힙: 가장 작은 값이 top에 오도록 설정
    std::priority_queue<int, std::vector<int>, std::greater<int>> pq;

    for (int i = 0; i < n * n; i++) {
        int num;
        std::cin >> num;

        pq.push(num);

        // 큐의 크기를 n개로 유지 (n개를 초과하면 가장 작은 놈을 제거)
        if (pq.size() > n) {
            pq.pop();
        }
    }

    // 결과적으로 큐에는 가장 큰 숫자 n개만 남고, top에는 그중 가장 작은(n번째로 큰) 숫자가 있음
    std::cout << pq.top() << "\n";

    return 0;
}

```
제미나이 사용 -> priority_queue 사용 방법 익히기
