# 풍선 터뜨리기

### [ 문제 정보 ]
> **백준**: https://www.acmicpc.net/problem/2346
> 
> **난이도**: 1
>
> **유형**: #자료구조


### [ 문제 풀이 ]
```C++
#include <iostream>
#include <deque>
#include <vector>

using namespace std;

int main() {
    int n;
    cin >> n;
    deque<pair<int, int>> dq; // {값, 번호}

    for (int i = 1; i <= n; i++) {
        int val;
        cin >> val;
        dq.push_back({val, i});
    }

    while (!dq.empty()) {
        int jump = dq.front().first;
        cout << dq.front().second << " ";
        dq.pop_front(); 

        if (dq.empty()) break; 

        if (jump > 0) {
            for (int i = 0; i < jump - 1; i++) {
                dq.push_back(dq.front());
                dq.pop_front();
            }
        } else {
            for (int i = 0; i < abs(jump); i++) {
                dq.push_front(dq.back());
                dq.pop_back();
            }
        }
    }
    return 0;
}
```
