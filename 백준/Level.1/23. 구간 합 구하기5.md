# 구간 합 구하기5

### [ 문제 정보 ]
> **백준**: https://www.acmicpc.net/problem/11660
> 
> **난이도**: 1
>
> **유형**: #동적계획법 (Dynamic Programming)


### [ 문제 풀이 ]
```C++
#include <bits/stdc++.h>
#include <cmath>
using namespace std;


int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);
    int N,M;
    cin >> N >> M;
    int dp[N+1][N+1] ;
    for (int i= 0 ; i<=N ; i++) {
        for (int j= 0 ; j<=N ; j++) {
            dp[i][j]=0;
        }
    }
    for (int i = 1 ; i <= N ; i++) {
        for (int j = 1; j<=N ; j++) {
            int ar ; 
            cin >> ar;
            dp[i][j] = dp[i-1][j] + dp[i][j-1] + ar - dp[i-1][j-1];
        }
    }
    for ( int i = 0 ; i<M ; i++) {
        int x1, y1, x2, y2;
        cin >> x1 >> y1>>x2>>y2;
        // cout << dp[x2][y2] << " "<< dp[x1][y1] << endl;
        cout<< dp[x2][y2] - dp[x1-1][y2] - dp[x2][y1-1] + dp[x1-1][y1-1] << "\n";
    }
}

```
