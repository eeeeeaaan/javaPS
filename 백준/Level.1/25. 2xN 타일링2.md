# 2xN 타일링2

### [ 문제 정보 ]
> **백준**: https://www.acmicpc.net/problem/11727
> 
> **난이도**: 1
>
> **유형**: #동적계획법 (Dynamic Programming)


### [ 문제 풀이 ]
```C++
#include <bits/stdc++.h>
#include <cmath>
using namespace std;


int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);
    int N;
    cin >> N;
    long long dp[N+1];
    dp[1] =1;
    dp[2] = 3;
    if (N>=3) {
        for (int i=3; i<=N; i++) {
            dp[i] = (dp[i-1] + 2*dp[i-2])%10007;
        }
    }

    cout << dp[N] << endl;

}

```
