# 스티커

### [ 문제 정보 ]
> **백준**: https://www.acmicpc.net/problem/9465
> 
> **난이도**: 1
>
> **유형**: #동적계획법 (Dynamic Programming)


### [ 문제 풀이 ]
```C++
#include <bits/stdc++.h>
#include <cmath>
using namespace std;


int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);
    int N;
    cin >> N;
    for (int i=0; i<N; i++) {
        int n ;
        cin >> n;
        vector<vector<long long>> original(3, vector<long long>(n+1) );
        vector<vector <long long>> dp(3, vector<long long>(n+1,0));
        for (int k =1; k<=2; k++) {
            for (int j = 1; j<=n ;j++) {
                cin >> original[k][j];
            }
        }

        dp[1][1] = original[1][1];
        dp[2][1] = original[2][1];
        for (int k =2 ;k<=n ;k++) {
            dp[1][k] = max(dp[2][k-1], dp[2][k-2])+ original[1][k];
            dp[2][k] = max(dp[1][k-1], dp[1][k-2])+ original[2][k];
        }
        cout << max(dp[1][n], dp[2][n]) << endl;

    }
}

```
