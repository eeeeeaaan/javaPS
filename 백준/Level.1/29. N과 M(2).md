# N과 M(2)

### [ 문제 정보 ]
> **백준**: https://www.acmicpc.net/problem/15650
> 
> **난이도**: 1
>
> **유형**: #백트래킹


### [ 문제 풀이 ]
```C++
#include <bits/stdc++.h>
#include <cmath>
using namespace std;


int n, m;
vector<int> current;
void recursive(int cnt, int current_number) {
    if ( cnt ==m ) {
        for ( int i=0; i<m; i++ ) {
            cout << current[i] << " ";
        }
        cout << "\n";
        return;
    }
    for (int i = current_number ; i <= n ; ++i) {
        
            current.push_back(i);
            recursive(cnt+1, i+1);
            current.pop_back();
        
    }
}
int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);
    cin >> n >> m;


    recursive(0, 1);
    return 0;


}
```
