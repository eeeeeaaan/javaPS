# N과  M(4

### [ 문제 정보 ]
> **백준**: https://www.acmicpc.net/problem/15652
> 
> **난이도**: 1
>
> **유형**: #백트래킹


### [ 문제 풀이 ]
```C++
#include <bits/stdc++.h>
#include <cmath>
using namespace std;


int n, m;
vector<int> current;
void recursive(int cnt, int current_number) {
    if ( cnt ==m ) {
        for ( int i=0; i<m; i++ ) {
            cout << current[i] << " ";
        }
        cout << "\n";
        return;
    }
    for (int i = current_number ; i <= n ; ++i) {
        current.push_back(i);
        recursive(cnt+1,i);
        current.pop_back();
    }
}
int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);
    cin >> n >> m;


    recursive(0, 1);
    return 0;


}

```
어제 푼거 문자 하나 변형하면 풀림 처음에 <br>recursive(cnt+1, i);<br>recursive(cnt+1, i+1); <br>했더니 중복이 너무 많이 나옴<br>왜 i인것만 남긴 거냐? 하면 중복 허용이 되기 때문에 꼭 현재 숫자 초과의 숫자를 다음에 넣을 이유가 없음 <br>이 다음에 들어가는 것 : 현재 숫자 이상 -> recursive 후 달라지는 것 i, i+1 인데 i는 곧 current_number 를 의미하고 ... 아무튼 그러함
