# 수들의 합4

### [ 문제 정보 ]
> **백준**: https://www.acmicpc.net/problem/2015
> 
> **난이도**: 2
>
> **유형**: #자료구조


### [ 문제 풀이 ]
```C++
#include <bits/stdc++.h>
#include <sstream>
using namespace std;


int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);
    long long n,k;
    cin >> n>> k;

    long long cnt=0; // 최종 출력 값
    long long pibo = 0 ; // 현재 누적 값
    map<long long, long long> countMap; // 같은 누적합이 몇개 나오는지 누적합을 key로 value는 누적합 나온 횟수
    for (int i=0 ; i<n;i++) {
        long long token;
        cin >> token;
        pibo += token;
        if (countMap.find(pibo-k) != countMap.end()) {
            cnt+= countMap[pibo-k];
        }
        if (pibo==k) {
            cnt++;
        }
        countMap[pibo]++;
    }
    cout << cnt << "\n";

}
```
