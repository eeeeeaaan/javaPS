# 이중 우선순위 큐

### [ 문제 정보 ]
> **백준**: https://www.acmicpc.net/problem/7662
> 
> **난이도**: 2
>
> **유형**: #자료구조


### [ 문제 풀이 ]
```C++
#include <bits/stdc++.h>
#include <sstream>
using namespace std;


int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);
    long long n;
    cin >> n;

    for (int i =0 ; i<n ;i++) {
        int instructionCount;
        cin>> instructionCount;

        map<long long, int> priorityQueue ;
        for (int j =0 ; j<instructionCount ; j++) {
            string instruction ;
            long long number;
            cin >> instruction >> number;
            if (instruction == "I")
                priorityQueue[number]++;
            else { // 삭제 연산 케이스
                if (!priorityQueue.empty()) {
                    if (number==-1) { // 최소값 찾기
                        if(priorityQueue.begin()->second ==1)
                            priorityQueue.erase(priorityQueue.begin()-> first);
                        else
                            priorityQueue.begin()->second--;
                    } else { // 최댓값 찾기
                        if (priorityQueue.rbegin()-> second ==1) {
                            priorityQueue.erase(priorityQueue.rbegin()->first);
                        } else {
                            priorityQueue.rbegin()->second--;
                        }
                    }
                }
            }
        }
        if (priorityQueue.empty()) {
            cout << "EMPTY"<<"\n";
        } else {
            long long min = priorityQueue.begin()->first;
            long long max = priorityQueue.rbegin()->first;
            cout << max <<" " << min <<"\n";
        }
    }
}
```
30분컷<br>map 과 set에서 key값을 기준으로 하면 최대최소 O(1)로 구할 수 있음<br>
