# 평범한 배낭

### [ 문제 정보 ]
> **백준**: https://www.acmicpc.net/problem/12865
> 
> **난이도**: 2
>
> **유형**: #동적계획법 (Dynamic Programming)


### [ 문제 풀이 ]
```C++
#include <iostream>
#include <map>
using namespace std;



int main() {
    std::ios_base::sync_with_stdio(false);
    std::cin.tie(NULL);

    int N, K;
    cin >> N >> K;
    long long dp[N+1][K+1];
    for (int i = 0 ; i<=K; i++) {
        dp[0][i] = 0 ;
    }
    for (int i = 1 ; i<= N ; i++) {
        int W, V;
        cin >> W>> V;

        for (int j = 0 ; j<= K; j++) {
            if (j >= W) {
                // 물건을 넣을 수 있는 무게라면?
                // "안 넣을래(위의 값)" vs "넣고 남은 무게 채울래(현재가치 + dp[이전][남은무게])" 중 큰 것!
                dp[i][j] = max(dp[i-1][j], V + dp[i-1][j - W]);
            } else {
                // 너무 무거워서 못 넣으면?
                // 그냥 윗집 값 그대로 가져오기
                dp[i][j] = dp[i-1][j];
            }

        }
    }
    cout<< dp[N][K] << endl;
}

```
