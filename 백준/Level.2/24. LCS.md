# LCS

### [ 문제 정보 ]
> **백준**: https://www.acmicpc.net/problem/9251
> 
> **난이도**: 2
>
> **유형**: #동적계획법 (Dynamic Programming)


### [ 문제 풀이 ]
```C++
#include <iostream>
#include <string>
#include <algorithm> // max 함수 사용

using namespace std;

// 최대 길이인 1000에 맞춰 넉넉하게 전역 선언 (자동 0 초기화)
int dp[1005][1005];

int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);

    string s1, s2;
    cin >> s1 >> s2;

    int s1len = s1.length();
    int s2len = s2.length();

    for (int i = 1; i <= s1len; i++) {
        for (int j = 1; j <= s2len; j++) {
            if (s1[i - 1] == s2[j - 1]) {
                // 1. 문자가 같으면: 대각선 왼쪽 위 + 1
                dp[i][j] = dp[i - 1][j - 1] + 1;
            } else {
                // 2. 문자가 다르면: 왼쪽(dp[i][j-1])과 위쪽(dp[i-1][j]) 중 큰 값
                dp[i][j] = max(dp[i][j - 1], dp[i - 1][j]);
            }
        }
    }

    // 최종 결과는 표의 맨 마지막 칸
    cout << dp[s1len][s2len] << "\n";

    return 0;
}
```
